# 3D Model Directory

Place your `SneakerModel.glb` file in this directory.

## Model Requirements

The GLB/GLTF model should have properly named mesh parts for color customization:

- **Laces** - Shoe laces (nodes named: Laces, laces, Lace, lace)
- **Logo** - Brand logo (nodes named: Logo, logo, Brand, brand)
- **SoleUpper** - Upper outsole (nodes named: SoleUpper, soleUpper, UpperSole, upperSole, Outsole, outsole)
- **SoleLower** - Midsole (nodes named: SoleLower, soleLower, LowerSole, lowerSole, Midsole, midsole)
- **Heel** - Heel section (nodes named: Heel, heel, HeelCup, heelCup)
- **Tongue** - Tongue section (nodes named: Tongue, tongue)
- **UpperBody** - Main upper body (nodes named: UpperBody, upperBody, Upper, upper, Body, body)
- **Stitching** - Stitching/seams (nodes named: Stitching, stitching, Stitch, stitch, Seam, seam)
- **InnerPadding** - Inside lining (nodes named: InnerPadding, innerPadding, Lining, lining, Inside, inside)

## Example Model Sources

You can find free 3D sneaker models at:
- Sketchfab (with free license)
- TurboSquid
- Free3D

Make sure the model format is GLB or GLTF and has proper mesh naming for the parts listed above.
